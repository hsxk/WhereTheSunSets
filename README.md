# WhereTheSunSets

`WhereTheSunSets` 是一个可以直接在浏览器里使用的小工具，帮你查看某一天、某个地点的:

- 日出时间和方向
- 日落时间和方向
- 月升时间和方向
- 月落时间和方向
- 黄金时刻和蓝调时刻
- 太阳与月亮当天的大致轨迹

它最适合这些场景:

- 想知道太阳会不会刚好落到山顶、塔尖或地标后面
- 想提前踩点拍日出、日落、月升、月落
- 想快速查看某一天某个地点的方位变化
- 想把某个日期和地点分享给别人一起看

## 特色

- 纯前端运行，不需要后端
- 可直接部署为静态网页
- 支持地图选点和浏览器定位
- 支持多语言界面
- 支持分享当前日期和位置
- 支持摄影场景下的目标对齐分析

## 怎么使用

打开页面后，你可以这样操作:

1. 点击“定位”按钮，或直接在地图上点一下，选择观察位置
2. 通过日期输入框、左右按钮或滑块切换日期
3. 在页面上查看日出、日落、月升、月落的时间和方向
4. 看地图上的方向线，判断太阳或月亮会从哪个方向出现或落下
5. 如果你是为了摄影踩点，可以展开高级功能，继续选择目标点，看是否会出现对齐时刻

## 高级功能

高级模式适合拍摄日落压山、日出穿塔、月升对地标这类画面。

使用方式:

1. 先设置观察点
2. 再设置目标点
3. 工具会给出目标方位、距离，以及接下来可能出现对齐的日期

需要注意:

- 这里的分析主要基于方位角
- 它适合做前期筛选和踩点参考
- 不等于真实地形遮挡计算，也不等于最终肉眼可见结果

## 本地打开

这个项目没有构建步骤，最简单的方式就是直接打开 [index.html](/Users/haokexin/service/WhereTheSunSets/index.html)。

如果你想完整体验定位、复制链接等浏览器能力，建议使用本地静态服务:

```bash
python3 -m http.server 8080
```

然后访问:

```text
http://localhost:8080
```

## 已知说明

- 所有结果仅供参考
- 天文计算基于 SunCalc
- 未接入真实山体轮廓、建筑遮挡或 DEM 地形数据
- 在高纬度地区，某些日期可能没有日出或日落
- 如果外部 CDN 或地图服务不可访问，部分功能会受影响
